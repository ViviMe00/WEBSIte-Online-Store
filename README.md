# WEBSIte-Online-Store
An online store made for a client.

# WordPress Online Store This online store was developed for a client using **WordPress**, focusing on performance, practicality, and full resource integration. 

## Main Features - **Bank integrations** via plugins optimized for online payments. 
- **Delivery system** directly connected to **Correios** (Brazilian postal service), ensuring automatic shipping cost calculation.
-  Full implementation of **WooCommerce**, enabling product, order, and customer management.
-  Use of the **Gutenberg editor** for creating and customizing pages.

## Website Structure
- *Home* — homepage with highlights and calls to action.
- *Store* — product display with filters and category organization.
- *Cart* — review and update selected items.
- *Checkout* — integrated and secure checkout.
- *My Account* — exclusive area for tracking orders and personal information.

 ---
  *Result:* a functional, modern, and easy-to-manage e-commerce site, offering a complete experience for both the customer and the end user.
